CSync class

this is just a collection of hints and tips. You should open util_sync.h and study interface :)

simple util, use SCOPE_LOCK macro to make entire function threadsafe or use CCriticalSection for custom locking

TryLock() is not supported under windows by default. To enable it, see util_sync.cpp

  • SCOPE_LOCK(cs) for locking whole function, it takes CCriticalSection instance as argument. It locks it instantly and unlock upon function return
  • Lock() Unlock() functions for CCriticalSection
  • Thread can lock critical section more than once, no harm done. Just unlock it same number of times
  • be aware how do you declare CCriticalSection instance, local variable won't do the trick. use global/member/static variable

See sync example for usage
