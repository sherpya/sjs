koraX's utils 1.7.0.0

Collection of simple and fast utilities to help you in your developement. These utilities were designed to work as a part of a Half-Life plugin (bot), so they have to be damn fast :)

This help is very brief, study examples and source code. Read change.log for more info, and readme.txt of course

You must know how to program under C or C++

Newest version of koraX's utils can be found on http://neuron.tuke.sk/~wagner

Copyright © 2005 Jozef Wagner, http://neuron.tuke.sk/~wagner

Features

  • precise runtime profiler, based on CPU cycles with low overhead (30 instructions)
  • loging to .xml file, redirecting cerr and cout to file
  • thread synchronization support
  • .xml/.ini/.cfg parser
  • 3D Vector, position and angle, with useful functions. Compatible with Half-Life's Vector class.
  • Random number generator, both float and integer
  • fast tests for floats, floats equality and zero test
  • and more (detect CPU speed, obtaining process and thread id, print ISO format date/time, thread creations, string conversions)

Requirements

These utilities use standard C++, so you must have decent C++ compiler :

  • Microsoft Visual C++ .NET
  • MinGW / DevC++
  • gcc

note that Visual C++ 6.0 is not supported, nor recommended. It is buggy and does not support standard C++ well.

Windows and linux platform are supported and tested. Any SuS compliant platform should work.

64 bit platform should work

Info

Quick how to

  1. source code is in directory /util
  2. copy it all somewhere into your source code
  3. include all util files into your project so compiler will compile them
  4. include util.h in your source code
  5. done, you can use all utils

All utilities are inside util namespace

Utilities

There are 7 main utilities

  • CSys - profiler and system
  • CLog - loging
  • CSync - thread synchronization
  • CThread - threads
  • CPar - xml/ini/cfg parser
  • CVector - 3d Vector, position, angle
  • misc - miscelaneous utils
